# CrimeWatch

**CrimeWatch** is a desktop application designed to help **undercover law enforcement officers manage investigation contacts efficiently and securely**.

### Target User
- Undercover law enforcement officers conducting ongoing investigations.
- Officers who need to track multiple contacts, encounters, and relationships during operations.

### What CrimeWatch Helps With
CrimeWatch centralizes investigation information to help officers stay organized and avoid losing important details.

Key capabilities include:
- Managing **contact profiles** related to investigations
- Recording **encounter logs** during field operations
- Tracking **relationships between individuals**
- Monitoring **follow-up statuses** for ongoing leads

### Value Proposition
CrimeWatch helps investigators:
- Keep all investigation-related information **organized in one place**
- Perform **rapid field logging** of encounters and notes
- Maintain **clear relationship links** between contacts
- Reduce **missed leads and duplicated work**
- Ensure **confidential and traceable investigation records**

### Project Context
This project is developed as part of a **Software Engineering course**, serving both as:
- A functional investigation management tool
- A codebase for studying and extending a real-world software project
